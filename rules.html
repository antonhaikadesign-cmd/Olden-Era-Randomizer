How Multiplayer Works

Short reference for the shared room flow and for the unlimited single-player mode.

Back to Randomizer

English

Single mode keeps the current sandbox experience: unlimited castle rerolls, unlimited hero rerolls, and unlimited coin flips.

Multiplayer mode creates a shared room and gives you an invite link. Send that link to your friend so both of you join the same matchup.

`Reroll Castles` is shared by both players and can be used only once total for the room.

Each player can use `Reroll Both Heroes` once. This rerolls both sides together but does not change the castles.

Each player can also use `Reroll Hero` once for their own side only. You cannot reroll your opponent’s hero.

`Coin 0/1` is shared. When any player flips it, the result updates for both players in the room.

Русский

В одиночном режиме всё остаётся как сейчас: бесконечный реролл замков, бесконечный реролл героев и бесконечный подброс монетки.

В мультиплеере создаётся общая комната и уникальная ссылка. Отправь эту ссылку другу, чтобы вы оба вошли в один и тот же матчап.

`Reroll Castles` — общая кнопка для двух игроков. Её можно использовать только один раз на всю комнату.

У каждого игрока есть по одному использованию `Reroll Both Heroes`. Эта кнопка меняет обоих героев сразу, но не меняет замки.

У каждого игрока есть также одно использование `Reroll Hero` только для своей стороны. Героя соперника рероллить нельзя.

`Coin 0/1` — общий функционал. Если любой игрок нажимает на монетку, результат сразу обновляется у обоих.
